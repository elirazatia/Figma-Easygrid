Using The Plugin Colmun / Row Values:
Can be in one of the following formats:
  — A single number to create {X} many columns / rows → '4'
  — A list of numbers, seperated by a space to create cells of fractional proportions → '1 3 1'

Tips for Column / Row Values:
  — Add 'pt' after a number to create a fixed size, unproprtional to the size of the selected layer → '25pt'
  — You can add '*' followed by another number to create multiple of the given cell → '25pt*2'

Gap Values:
Enter any number, including negative.
